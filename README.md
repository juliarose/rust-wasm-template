# rust-wasm-template
Template for building Rust WASM projects.

## Setup
- Install wasm-pack  
```console
curl https://rustwasm.github.io/wasm-pack/installer/init.sh -sSf | sh
```
- Install node modules  
```console
npm install
```
- Build WASM  
```console
npm run build-wasm
```
- Start express server  
```console
npm run serve
```
